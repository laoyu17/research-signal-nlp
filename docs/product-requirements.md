# Product Requirements - research-signal-nlp

## 1. 目标
面向量化研究实习面试，构建一个可复现的文本信号研究框架：
- 输入：公告/研报/新闻类文本
- 输出：可用于回测的日频横截面因子 + 事件信号
- 验证：IC、分层收益、事件窗 CAR、稳定性

## 2. 用户画像
- 主用户：求职中的量化研究/量化开发候选人
- 次用户：需要快速验证文本因子的研究同学

## 3. 功能范围
### 必须
1. 数据导入与字段标准化
2. 规则词典情绪与事件抽取
3. TF-IDF + 线性模型 baseline
4. 横截面回测与事件研究
5. PyQt6 研究工作台与报告导出

### 可选增强
1. Transformer 推理/微调接口
2. 行业中性化、风格中性化增强
3. 多数据源适配器

## 4. 非功能要求
- 可复现：示例配置一键运行
- 可解释：每个指标可追溯到原始输出
- 可扩展：抽象出 extractor/model/backtest 接口
- 可演示：本地 GUI 全流程可视化

## 5. 成功指标
- 示例数据全链路运行成功
- 自动报告可生成并包含关键图表
- 核心测试通过率 100%
