# Interview Highlights

## 30 秒电梯介绍
我做了一个开源文本因子研究框架，把公告/研报文本转成可回测信号，支持横截面和事件研究双轨评估，并提供桌面研究工作台和自动报告。

## 2 分钟技术亮点
1. 先做强基线：规则词典 + 事件抽取 + TF-IDF 线性模型，确保可解释和可复现。
2. 信号输出是标准化因子分数，直接可用于 IC、分层收益与稳定性评估。
3. 提供 GUI 和 CLI 双入口，面试可演示，研究可复用。
4. 工程化具备 CI、测试分层与回归阈值管理。

## 可扩展方向
- 替换为中文金融预训练模型
- 接入商业文本数据源
- 增加行业/风格中性化模块
